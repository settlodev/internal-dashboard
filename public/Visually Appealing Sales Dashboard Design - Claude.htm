    What does change refer

    In the context of the dashboard, the "change" value represents the percentage increase or decrease in a particular metric compared to a previous period (typically the previous month or previous time frame).

    For example:

    • In change: -74.2, the negative value indicates a 74.2% decrease
    • In change: 35, the positive value indicates a 35% increase

    So in the performance metrics:

    javascript
    {
  name: 'Total Revenue',
  value: salesData.totalRevenue.toLocaleString(),
  change: -74.2,  // This means total revenue decreased by 74.2%
}

    The change is visually represented by:

    1. A directional arrow (↑ for increase, ↓ for decrease)
    2. The absolute value of the percentage
    3. Color coding (red for negative, green for positive)

    This helps viewers quickly understand the trend of key business metrics - whether they're improving or declining, and by how much.
